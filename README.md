# Healthcare-Analysis-
![Healthcare Dashboard](aayushitejwani7.png)
The Healthcare Monitoring Dashboard is a data visualization project designed to analyze and monitor key healthcare metrics using interactive and insightful dashboards. The goal of this project is to transform raw healthcare data into meaningful insights that help in understanding patient trends, medical resource distribution, and allergy patterns.

This dashboard provides a comprehensive overview of healthcare data by displaying important Key Performance Indicators (KPIs) such as the total number of patients, the total number of doctors, and the total number of recorded allergies. These KPIs provide a quick snapshot of the healthcare system's overall status and help stakeholders understand the scale of patient records and medical resources available.

The dashboard also includes multiple visualizations that allow users to explore the data in a more detailed and interactive manner. A city-wise patient distribution chart shows how patients are distributed across different cities, making it easier to identify which locations have the highest concentration of patients. This information can help healthcare organizations understand regional demand and allocate resources effectively.

Another key component of the dashboard is the allergy distribution chart, which highlights the proportion of patients suffering from different allergies. By visualizing this data, healthcare professionals can identify the most common allergies among patients and understand how allergy-related cases contribute to overall patient statistics.

The dashboard also includes a time-series analysis of allergies by year, which helps track how allergy cases have changed over time. This visualization enables users to detect trends, patterns, and fluctuations in allergy occurrences across different years, supporting better healthcare planning and decision-making.

In addition, the patients by year chart provides insight into the number of patient admissions over time. By analyzing these trends, healthcare administrators can evaluate whether patient numbers are increasing or decreasing and plan accordingly for future healthcare demands.

To enhance usability and interactivity, the dashboard includes filter options such as province and allergy type, allowing users to dynamically explore the data and focus on specific segments. This makes the dashboard highly flexible and useful for both high-level overviews and detailed data analysis.

Overall, this project demonstrates the power of data analytics and visualization in healthcare, enabling users to identify patterns, monitor trends, and make data-driven decisions. It highlights how analytical tools can improve healthcare insights by converting complex datasets into clear and actionable visual information.

Tools and Technologies Used

Power BI for data visualization and dashboard creation

Healthcare dataset for analysis and reporting

Data modeling and transformation techniques

Interactive filters and slicers for dynamic insights
